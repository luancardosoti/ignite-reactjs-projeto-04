### Requirements

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/) or [npm](https://www.npmjs.com/package/npm)

**Clone the project and access the folder**

```bash

```

**Follow the steps below**

```bash
# Install the dependencies
$ npm i
# Run project
$ npm run dev
```
